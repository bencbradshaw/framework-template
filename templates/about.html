About Framework Template

A comprehensive starter template for building modern web applications with Go and TypeScript.

Straightforward way to create public HTML pages

Simplistic development of SEO friendly public web pages. Gone are the days of playing games with JavaScript rendering engines and frameworks. Simply write HTML for a public page.

Create 1 or multiple single page applications

Easily handle routing to 1 or more isolated single-page-applications, your JavaScript app. Without any node process, this framework will handle all JavaScript watching and bundling for you, thanks to Esbuild's native go support.
